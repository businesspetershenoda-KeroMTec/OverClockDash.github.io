About The Game

Thrive on the Grid

Built on Unity's high-performance 3D engine, OverClockDash: Runner challenges players to survive within a dynamically generated procedural synthwave environment. As you progress, the core processing units overclock, constantly raising game speed and raising stakes.

Master precision aerial flips, slide beneath low-clearance hyper-truck barricades, and utilize advanced power-up cores to break through defense grids in a flawless loop of intense reaction-based action.
